Este portafolio está construido con tecnologías de última generación para garantizar rendimiento, SEO y una experiencia de usuario técnica.

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
